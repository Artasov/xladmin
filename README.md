# xladmin

`xladmin` это монорепа с двумя пакетами для админки _like django_ поверх `FastAPI + SQLAlchemy + Pydantic + Next`.

- backend: [pypi.org/project/xladmin](https://pypi.org/project/xladmin/)
- frontend: [npmjs.com/package/xladmin](https://npmjs.com/package/xladmin)

Что есть из коробки:

- список моделей и overview
- список объектов, поиск, сортировка и пагинация
- create / update / delete
- bulk actions и object actions
- модельные блоки в sidebar и overview
- RU / EN locale
- форматирование `date` и `datetime`
- preview удаления связанных объектов перед delete и bulk delete
