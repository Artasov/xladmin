# xladmin

`xladmin` это монорепа с двумя пакетами для админки поверх `FastAPI + SQLAlchemy + Pydantic + Next`.

- backend: PyPI [`xladmin`](https://pypi.org/project/xladmin/)
- frontend: npm [`xladmin`](https://www.npmjs.com/package/xladmin)
- репозиторий: [Artasov/xladmin](https://github.com/Artasov/xladmin)

Что есть из коробки:

- список моделей и overview
- список объектов, поиск, сортировка и пагинация
- create / update / delete
- bulk actions и object actions
- модельные блоки в sidebar и overview
- RU / EN locale
- форматирование `date` и `datetime`
- preview удаления связанных объектов перед delete и bulk delete
